# how to run project?
1- create virutal env
```
  pythonx -m venv .venv
&copy;
```
2- activate virutal env
  ```
  source .venv/bin/activate
&copy;
```
3- install dependencies 
  ```
  pipx install -r requirements.txt
&copy;
```
4-  creating database 
  ```
  ./manage.py makemigrations
  ./manage.py migrate
&copy;
```
